Microfix Workspace v1.0.0-alpha

Microfix is a high-performance, purely Go-based FIX protocol engine and debugging workstation.

Core Concepts

The Stream

The heartbeat of Microfix. It acts like a Wireshark capture for FIX. Messages are color-coded (Green = Executions, Blue = Business, Gray = Admin, Red = Rejects).

Contextual Inspector

Clicking any message in the stream slides out the Inspector. Use this to view parsed repeating groups, debug sequence errors, or diff two payloads.

Script Runner (.mxs)

Microfix supports an internal batch scripting language. You can automate Logon sequences, Order flows, and expect/assert specific FIX responses without writing external code.

Built with Go, Wails, and Alpine.js.
